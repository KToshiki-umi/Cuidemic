[説明]
  1人プレイ用のコンソール版「Pandemic」です。検疫官、危機管理官も追加しています。
  動作環境はPython 2.7.6になります。
  ゲームはmain.pyを実行して開始してください。


[プレイ上の注意]
  殆どの操作は、何も入力せずにEnterを押すことでキャンセルできます。
  カードや都市の選択などで「03」とある場合に、「03」、または「3」と入力することで選択できます。
